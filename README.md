# Space-Shooter

Original project (Unity 4): http://unity3d.com/learn/tutorials/projects/space-shooter/

Adapted for Unity 5 and customized.


# Controls:

Option 1)
 
  To move: Arrow keys
  
  To shoot: Space bar
  
Option 2)
 
  To move: W (up), A (left), D(right), S(down)
  
  To shoot: Mouse left click
  

Enjoy!
