# Space-Shooter

Original project (Unity 4): http://unity3d.com/learn/tutorials/projects/space-shooter/

Adapted for Unity 5 and customized.


# Controls:

Option 1)
 
  Moving: Arrow keys
  Shooting: Space bar
  
Option 2)
 
  Moving: W (up), A (left), D(right), S(down)
  Shooting: Mouse left click
  
Enjoy!
